## Clone This Project

```
git clone https://github.com/SyaPratama/Karyawan-MVC.git
```

```
cd Karyawan-MVC
```

```
composer update
```

```
cd public
php -S 127.0.0.1:8080
```

> Project Karyawan Sudah Bisa Dijalankan
